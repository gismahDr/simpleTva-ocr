# Impact Analysis

Use this skill before modifying entities, DTOs, services, APIs, database schemas, or shared components.

## Goal

Understand all downstream effects before changing code.

## Checklist

Before editing:

* Find all references.
* Identify dependent modules.
* Review related APIs.
* Review frontend usage.
* Review database impact.
* Review integrations.

## Spring Boot Checklist

Check:

* Entities
* Repositories
* Services
* Controllers
* DTOs
* Validation rules
* Security configuration

## Frontend Checklist

Check:

* Vue components
* API calls
* Forms
* Tables
* Reports

## Deliverable

List:

* Files impacted
* Risks identified
* Validation required
